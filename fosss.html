here is some free and open source softwares

development:

  • notepad++ & notepadqq - simple but powerful text and code editor
  • geany - simple ide
  • vscodium - floss binaries of vs code
  • kate - ide like text editor
  • gedit - powerful text editor
  • micro - a easy to use terminal text editor

graphics:

  • krita - designed for artist but can be used for basic photo editing
  • photogimp - a modification of gimp that transforms its interface into one similar to photoshop's
  • pinta - paint.net alternative.
  • blender - hard to learn but really powerful 3d creation software

productivity:

  • okular - document viewer and editor
  • superproductivity - todo list with some extra features
  • libreoffice - alternative to office 365
  • onlyoffice - best compatible alternative to office 365
  • thunderbird - mail client that includes calender, rss feed, adress book, newsgroups and chat features

utilities:

  • filezilla - ftp client
  • gnome disk utility - my favorite disk partition tool for linux
  • bottles - really good app that allows you to run windows apps on linux with wine really easily
  • peazip - best winrar alternative
  • ventoy - create usb flash drives that can boot into different isos
  • kde connect - integrate your phone to your computer
  • converter - local image converter
  • easy effects - audio effects for pipewire applications
  • handbrake - local video converter
  • virtualbox - virtualization tool that good at everything except gaming
  • transmission - lightweight torrent client
  • openssh - connect to another computers command line
  • tigervnc - connect to another computer

multimedia:

  • vlc - simple and useful media player
  • obs studio - not very simple but very powerful video recording and live streaming app
  • vokoscreenng - very simple and useful and powerful video recorder software
  • shotcut - easy and useful video editor
  • tenacity - simple but powerful audio editor
  • kdenlive - simple and powerful video editor
  • rhythmbox - audio player
  • cheese - take photos and videos with you webcam

security:

  • clamav & clamtk - antivirus
  • bitwarden - online and free synchronized password manager
  • keybase - encrypted open source cloud file storage utility that has 250gb free storage

gaming:

  • prism launcher - a really good Minecraft client that forked from multimc
  • vinegar - roblox wine wrapper with some extra features for gaming
  • lutris - a really good game manager

browsers:

  • floorp - a firefox fork that have improvements and some customization options
  • epiphany - basic browser with a beatiful gui
  • firefox - web browser
  • waterfox - based on firefox
  • librewolf - a firefox fork that focuses on privacy
  • brave - chromium based privacy oriented browser(have some anti privacy features)
  • tor browser - dark web browser

internet:

  • freetube - youtube client(it is really good)

communication:

  • vencord - discord client
  • revolt - discord alternative that has nitro features for free
  • pidgin - msn looking chatting client that supports different protocols including but not limited to irc and xmpp
  • element - matrix client. very secure

android apps:

  • vendroid - discord client
  • jerboa - lemmy client
  • element
  • firefox
  • keybase
  • proton mail
  • bitwarden

  • seal - android yt-dlp gui frontend

favourite distros:

  • arch linux - gives you a base to build upon, and you can say "i use arch btw"
  • debian gnu/linux - rock solid stability
  • fedora linux - basically rhel but free and more up to date
  • artix linux - the unholy offspring of gentoo and arch; basically arch with a different init system
